Semantic Tags

Semantic tags are tags that are used to describe the content of a web page. They are used to provide meaning to the content of a web page. They are also used to improve the accessibility of the web page.
Here you will find a list with some of the most important semantic tags

  • Header
  • Section
  • Aside
  • Footer
  • Nav

Hyperlinks

Hyperlinks are used to link one web page to another. They are used to navigate between web pages. They are also used to provide a way to share information between different web pages.
